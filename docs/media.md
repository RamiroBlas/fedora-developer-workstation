# Multimedia

## Objetivo

Reducir el uso del navegador para escuchar música y consumir contenido.

## Herramientas

- mpv
- yt-dlp
- Yewtube

## Instalación

pipx install yewtube --backend pip

## Configuración
Dentro del Yewtube se coloca

- set player mpv
- set show_video false
- set max_res 720

## Uso
se inicia con yt en la terminal

## Para ver configuracion
- set

## Atajos mpv

Espacio → Pausa

q → Salir

9/0 → Volumen

← → Retroceder

→ → Avanzar
