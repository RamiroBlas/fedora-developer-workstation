# Fedora Developer Workstation

Configuración personal de Fedora KDE para desarrollo Full Stack, DevOps, IA y productividad keyboard-first.

## Objetivos

- Entorno reproducible.
- KDE Plasma minimalista.
- Flujo centrado en teclado.
- Terminal profesional.
- Automatización de proyectos.
- Configuración versionada en Git.
